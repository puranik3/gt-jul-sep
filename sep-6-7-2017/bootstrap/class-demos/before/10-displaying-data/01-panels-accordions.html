Displaying Data - Panels

Demo A : A panel is a component that looks like a window and has a heading and body.
.panel (base class) and .panel-[theme] for container
.panel-heading with optional .panel-title for heading
.panel-body for body

What is jQuery?

jQuery is a cross-browser compatible general-purpose JS library.

What is Bootstrap?

Bootstrap is a responsive, mobile-first, front-end framework.

Demo B : A panel can be made collapsible by wrapping the panel body within a .collapse.panel-collapse container. Add .in to make it visible by default. Make the heading/title a trigger for toggling the panel body by adding href/data-target (pointing to the collapsible container) and data-toggle="collapse".

What is Bootstrap?

Bootstrap is a responsive, mobile-first, front-end framework.

Demo C : Convert a series of collapsible panels into an accordion by adding a container .panel-group around the panels. Add data-parent="selector for panel group" to each trigger.
Note: When one panel in a group opens, the one currently open, closes.
